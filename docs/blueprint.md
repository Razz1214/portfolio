# **App Name**: Raj's Digital Canvas

## Core Features:

- Home Page: Showcase a brief introduction of Raj Kumar, his skills, and career aspirations.
- Projects Showcase: Display Raj's projects with descriptions, technologies used, and links to GitHub or live demos.
- Skills Overview: Visualize Raj's technical skills with proficiency levels.
- Education & Experience: Timeline showcasing Raj's education at Pimpri Chinchwad University and relevant work experiences.
- Contact Form: Allow visitors to send messages to Raj directly through the portfolio.
- AI-Powered Personalization Tool: Allow user to get personalizd welcome text that shows some basic facts about Raj Kumar using LLM as a tool to make decisions on text relevance.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to represent intelligence and focus.
- Background color: Light grey (#E0E0E0) to ensure readability and a modern feel.
- Accent color: Vibrant gold (#FFD700) to highlight key skills and achievements.
- Body and headline font: 'Inter', a sans-serif font, provides a modern, clean and highly readable feel for all text.
- Use a consistent set of minimalist icons to represent skills and technologies.
- Use a clean, modern, and responsive layout to ensure optimal viewing on all devices.
- Subtle animations to highlight key elements upon user interaction (e.g., skill bars filling up, project cards flipping).
